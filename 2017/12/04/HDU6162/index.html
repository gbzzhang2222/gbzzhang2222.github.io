HDU6162

发表于 2017-12-04

Description

给出一棵带点权的树，每次询问两点之间路径上权值在$[a_i,b_i]$之间的点权和。

Solution

不知道有没有什么巧妙的做法，反正我是工业了一波。

转有根树，记录每个点到根节点上所有点权的权值线段树，离散化之后可持久化。
在LCA处作差即可。

不知道为什么放到树链剖分里，然后就强行树剖写了一发LCA。
肯定是我太垃圾了。
